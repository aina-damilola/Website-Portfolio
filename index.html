Projects
Space-Dog
Learn More

Space-Dog utilizes C++ and one of its frameworks, the Arduino Library to create an autonomous Rover that follows the nearest Lightsource.

It uses photoresistors as eyes to locate the strongest lightsource, it also uses ultrasonic sensors to detect and avoid collision, the dual motors to allow the chasis to turn and a central Arduino NANO to control all these different parts.

SIMON
Learn More

Simon utilizes Verilog HDL and a De1-Soc FPGA Board, all integrated with the use of Intel Quartus Prime, to create a Piano Simon game.

The game uses different clock cycles from numerous peripherals (PS/2 Keyboard & Mouse, Speakers, VGA, Memory blocks) to allow for their signals to be properly integrated. Multiple FSMs were used to properly control these peripherals including the data & control path of the program.

WORLDLE
Learn More

This project utilizes the Python Programming Language and its Tkinter Library to create a New York Times 'Wordle' inspired game, Worldle.

The game compares the country inputed with a dictionary of countries showing the user which letters are correct or wrong, whether or not the target country has a higher letter count and whether or not they're both in the same continent.
